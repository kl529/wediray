## 앱 아이콘 가이드

**확정된 콘셉트:** 폴라로이드 프레임 안에 편지봉투(핑크 하트) + 다이아몬드 반지, 핑크 글리터 테이프, 블랙 배경, Y2K 키치

| 파일 | 크기 | 요구사항 |
|------|------|---------|
| `assets/icon.png` | 1024×1024px | 투명도 불가, 사각형 원본 (라운드는 iOS 자동 적용) |
| `assets/adaptive-icon.png` | 1024×1024px | 투명 배경, 그래픽은 중앙 66% 안에만 |
| `assets/splash-icon.png` | 1024×1024px | 투명 배경 OK, 블랙 배경 위에 올라감 |

**AI 생성 프롬프트:**
```
iOS app icon, Y2K diary aesthetic, polaroid frame containing a pink heart envelope
and diamond ring, hot pink glitter tape accent, black background, rounded square format,
clean flat design, no text, bold graphic, slightly retro digital feel, 1024x1024
```

**아이콘 적용 후 재실행:**
```bash
cd app && npx expo start --clear
```

## 제출 체크리스트

### 사전 준비
- [ ] `eas login` — EAS CLI 로그인
- [ ] `eas build:configure` — 프로젝트 초기화
- [ ] App Store Connect에서 앱 생성 (번들 ID: `com.lyvakim.wediary`)
- [ ] Apple Developer Program 가입 확인 ($99/년)
- [ ] `eas.json`의 `appleId`, `ascAppId`, `appleTeamId` 입력

### 에셋 준비
- [ ] 앱 아이콘 (`assets/icon.png`, 1024×1024px) 완성
- [ ] Android 어댑티브 아이콘 (`assets/adaptive-icon.png`) 완성
- [ ] 스플래시 아이콘 (`assets/splash-icon.png`) 완성
- [ ] 스크린샷 5장 완성 (iPhone 6.9인치 기준)

### 스토어 설정
- [ ] App Store Connect 가격 설정: **Tier 1 (₩1,200)** 으로 출시
- [ ] 개인정보처리방침 URL 호스팅 후 입력
- [ ] 지원 URL 또는 이메일 입력

### 빌드 & 제출
```bash
# 프로덕션 빌드
cd app && eas build --profile production --platform ios

# App Store 제출
cd app && eas submit --platform ios --latest
```
