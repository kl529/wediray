> 작성일: 2026-04-04 | 대상: v1.0 App Store 출시

## 1. 포지셔닝

**"받은 청첩장만큼의 소중한 기억들을, 링크 하나로."**

wediary는 결혼식에 *가는 사람(게스트)* 의 앱이다.
웨딩북·아이웨딩이 신랑신부 플래닝 도구라면, wediary는 연간 10개씩 결혼식에 불려다니는 2030의 개인 기록장이다.

| 앱 | 타겟 | 포지션 |
|----|------|--------|
| 웨딩북 | 신랑신부 | 웨딩 플래닝 도구 |
| 아이웨딩 | 신랑신부 | 업체 연결 플랫폼 |
| 데어무드 | 신랑신부 | 청첩장 제작 서비스 |
| **wediary** | **게스트** | **유일한 게스트 사이드 기록 앱** |

> 직접 경쟁자 없음. 마케팅 핵심 과제: "이런 앱이 있었어?" 라는 발견 경험 만들기.

## 2. 타겟 페르소나

### Primary — 결혼식 시즌 포화된 직장인 여성

| 항목 | 내용 |
|------|------|
| 이름/나이 | 지은, 28세 |
| 직업 | 마케터, 대기업 3년차 |
| 디지털 습관 | 인스타그램 데일리, 카카오톡 메인 소통 창구 |
| 결혼식 참석 빈도 | 연간 12~18개 |
| 핵심 페인포인트 | "저번에 얼마 냈지?", "그 식장 어디였더라?", "내가 참석을 했나?" |
| 감성 키워드 | Y2K, 폴라로이드, 다꾸, 복고감성 |
| 앱 발견 경로 | 인스타그램 릴스, 앱스토어 검색, 친구 추천 |

**앱을 쓰는 순간:**
- 카톡으로 청첩장 받은 직후 → 링크 붙여넣기
- 결혼식 당일 or 다음 날 → 한줄 소감 + 축의금 기록
- 다음 결혼식 선물 고민 중 → "지난번에 얼마 냈지?" 확인

### Secondary — 축의금 관리가 필요한 직장인 남성

| 항목 | 내용 |
|------|------|
| 이름/나이 | 민준, 31세 |
| 결혼식 참석 빈도 | 연간 8~12개 |
| 핵심 페인포인트 | 축의금 기록, 날짜·장소 확인 |
| hook | Y2K 감성보다 축의금/일정 관리 기능으로 어필 |

### Tertiary — 결혼 준비 중인 신부 예정자

참석한 결혼식을 레퍼런스로 모아두는 니즈. 단, 메인 메시지에서는 제외 — 웨딩 플래닝 앱으로 오해받을 수 있음.

## 4. 가격 전략

### 모델: 일시불 유료앱 (구독 없음)

구독은 비수기 해지 + 데이터 불안을 유발한다.
wediary는 "한번 사면 내 기록이 평생 내 것"이 더 강한 가치 제안이다.

### 2단계 가격 로드맵

| 시기 | 가격 | 목적 |
|------|------|------|
| 출시 ~ D+60 | **₩1,200** (Tier 1, $0.99) | 얼리버드 — 리뷰/다운로드 확보 |
| D+61 이후 | **₩3,900** (Tier 3, $2.99) | 정상 수익화 |
| 성수기 직전 (3월 초, 9월 말) | **₩2,500** (2주간) | 시즌 할인 |

가격 인상 전 인스타그램에 "출시 기념 가격 종료" 공지 1개 → 공지 자체가 마지막 구매 트리거.

### 프로모 코드 (리뷰 확보용)

- iOS: 버전당 최대 100개, 발급 후 28일 만료 (App Store Connect → 프로모션 코드)
- Android: 월 최대 500개 (Play Console → 수익 창출 → 프로모션 코드)
- **용도:** 지인 20명에게 무료 코드 배포 → 리뷰 10개 확보 → 광고 시작

### 수익 시뮬레이션 (정상가 ₩3,900 기준, 수수료 30% 제외)

| 시나리오 | 월 다운로드 | 월 순수익 |
|---------|-----------|---------|
| 보수적 | 100건 | ₩273,000 |
| 중간 | 300건 | ₩819,000 |
| 낙관적 | 1,000건 | ₩2,730,000 |

**v2 수익화 아이디어:** 사진 무제한 + 통계 + 연말 리포트 → 추가 일시불 업그레이드 ₩3,900

## 5. 마케팅 채널

운영 부담 없이 단기 집중. **채널 3개만** 쓴다.

| 순서 | 채널 | 목적 | 비용 | 기간 |
|------|------|------|------|------|
| 1 | 지인 프로모 코드 | 리뷰 10개 선확보 | 0원 | 출시 후 1주 |
| 2 | 인스타그램 릴스 광고 | 초기 다운로드 | ~21만원 | 2주 집중 후 종료 |
| 3 | 커뮤니티 자연 후기 | 오가닉 신뢰도 | 0원 | 광고와 동시 |

### 채널 1: 지인 프로모 코드

광고 전에 리뷰를 먼저 만든다. 리뷰 0개 상태에서 광고를 돌리면 전환율이 거의 0.

- 프로모 코드 20개 발급 → 지인/가족/동료에게 카톡 발송
- 목표: **리뷰 10개, 평점 4.0+** 달성 후 광고 시작

### 채널 2: 인스타그램 릴스 유료 광고

**타겟:** 여성, 24~34세, 한국, 관심사(결혼식·웨딩·다이어리·라이프스타일)

**광고 소재 — 영상 1개:**
- 15~30초 화면 녹화
- 흐름: 카카오톡 청첩장 링크 복사 → 앱 붙여넣기 → 자동 입력 완료 → 완성된 카드
- 텍스트: `"받은 청첩장만큼의 기억들"` → `"링크 하나로 3초 등록"`
- 마지막 프레임: 앱스토어 다운로드 버튼 + `₩1,200`

**예산:** 하루 1.5만원 × 14일 = **21만원**
2주 후 CPI 확인 → ₩5,000 이하면 1주 연장, 초과하면 종료.

### 채널 3: 커뮤니티 자연 후기 글 (1회)

- 더쿠 `자유게시판` 또는 인스티즈 `인티포커스`에 실사용 후기 글 1개
- 에펨코리아 `자유게시판` (남성 페르소나 커버)
- **주의:** 광고 글 형태 금지. "나 결혼식 많이 다니는데 이런 앱 씀" 형태로. 본문에 링크 직접 삽입 X.

## 6. 런치 플랜

### Phase 0: 출시 준비 (D-14 ~ D-1)

- [ ] 앱 아이콘 완성 (`assets/icon.png` 교체)
- [ ] 시뮬레이터 더미 데이터 세팅 후 스크린샷 5장 촬영
- [ ] 인스타그램 광고용 영상 1개 제작
- [ ] App Store Connect 가격 설정 (₩1,200, Tier 1)
- [ ] 프로모 코드 20개 발급

### Phase 1: 리뷰 확보 (D+0 ~ D+7)

- [ ] 앱 출시
- [ ] 지인 20명에게 프로모 코드 + 리뷰 요청 카톡 발송
- [ ] 목표: 리뷰 10개, 평점 4.0+ 확인 후 다음 단계 진행

### Phase 2: 광고 집행 (D+8 ~ D+21)

- [ ] 인스타그램 광고 시작 (하루 1.5만원)
- [ ] 더쿠/인스티즈 자연 후기 글 1개 게시
- [ ] D+14에 CPI 확인 → ₩5,000 이하면 1주 연장, 초과하면 종료

### Phase 3: 오가닉 유지 (D+22 이후)

- [ ] 앱스토어 리뷰 모니터링 + 답글
- [ ] D+60에 ₩3,900으로 가격 인상 + 인스타 공지 1개
- [ ] 성수기 전 시즌 할인 (3월 초, 9월 말) 2주간
